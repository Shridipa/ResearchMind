# API Reference

The canonical API documentation is generated by FastAPI at `/docs`.

## Core Endpoints

### `POST /api/v1/papers/upload`

Uploads a PDF, parses it, chunks it, embeds chunks, and indexes them.

### `POST /api/v1/chat`

Runs grounded retrieval QA over indexed papers.

### `POST /api/v1/papers/summarize`

Generates an abstract, contributions, methods, findings, limitations, and citations.

### `POST /api/v1/flashcards`

Generates question-answer flashcards from retrieved content.

### `POST /api/v1/compare`

Compares multiple indexed papers along method, dataset, contribution, limitation, and future-work dimensions.
